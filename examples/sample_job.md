# バックエンドエンジニア（Webサービス開発）

私たちは自社プロダクトを開発する成長中のスタートアップです。
サーバーサイドの設計・実装をリードしてくれる仲間を募集しています。

## 必須要件

- Python を用いたWebアプリケーション開発の実務経験（3年以上）
- Django もしくは FastAPI でのAPI開発経験
- PostgreSQL などRDBを用いた設計・運用経験
- Git によるチーム開発の経験
- REST API の設計経験

## 歓迎要件

- Docker / Kubernetes を用いたコンテナ運用の経験
- AWS もしくは GCP 上でのインフラ構築経験
- CI/CD パイプラインの構築経験
- GraphQL を用いたAPI開発の経験
- TypeScript / React によるフロントエンド開発の経験
